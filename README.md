An interactive platform (simulator) for teaching programming with automatic assignment checking.

Вижн-документ
Интерактивная платформа (тренажер) для обучения программированию с автоматической проверкой заданий
1. Общее описание
1.1 Цель проекта
Создать веб-платформу, позволяющую учащимся интерактивно изучать программирование (Python и SQL), выполнять задания прямо в браузере, получать автоматическую обратную связь и видеть свой прогресс.
Преподаватели смогут управлять курсами, видеть результаты, назначать дедлайны и получать данные по успеваемости.
2. Целевая аудитория
Студенты, обучающиеся программированию.
Преподаватели, ведущие курсы по Python/SQL.
Университет, желающий встроить систему автопроверки.
3. Проблемы, которые решает продукт
Отсутствие удобной среды для мгновенного запуска и проверки кода на питоне и SQL без установки ПО на онлайн курсах ВШЭ
Трудозатраты преподавателей на проверку заданий, как следствие задержка проверок и несвоевременная обратная связь (студенты могут забыть задание и логику решения за время проверки)
Недостаток мотивации к практике и обратной связи у студентов.
4. Ключевые особенности и возможности
Базовый функционал:
Реализация среды с автопроверкой заданий (кодовое окно, консоль, тесты)
Поддержка Python-заданий
Веб-интерфейс на базе Marimo, запуск кода в браузере
Подсказки и пояснения к ошибкам
Личный кабинет ученика и преподавателя
Электронная ведомость, сохранение прогресса
*более подробные функциональные требования по этапам 


Расширенный функционал:
Поддержка SQL-заданий
Лидерборды, система мотивации
Назначение домашних заданий, дедлайны
Темная/светлая тема
Дополнительный функционал:
AI-помощник по коду и ошибкам
Встроенные заметки
Экзаменационный режим с таймером
Аналитика для преподавателей
5. УТП
Автопроверка, мгновенная обратная связь и обучающая поддержка, всё в одном окне браузера, без необходимости установки IDE или окружения.
6. Ограничения и рамки
На старте поддерживаются только Python
Нет реального окружения - студенты скорее тренируют синтаксис, нежели пробую реальные задачи, которые могут быть на будущей работе
В рамках MVP реализуется только базовый функционал
